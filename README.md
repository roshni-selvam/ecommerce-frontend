# 🛍️ Trendy Vibe — Women's Fashion Store

A women's western wear e-commerce store with a elegant Rose Gold theme.

## Features
- 👗 Product listing — Tops, Dresses, Bottoms, Sets, Outerwear
- 🛒 Cart management with Context API
- 🔐 Login & Register with localStorage auth
- 📦 Product detail view
- 💳 Checkout flow
- 📱 Fully responsive design

## Tech Stack
- React + Vite
- Context API
- CSS
- localStorage

## Getting Started
```bash
npm install
npm run dev
```

## Live Theme
- 🎨 White + Grey + Rose Gold (`#b76e79`)
